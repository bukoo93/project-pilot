# Output Format Standards

## Language Handling
All commands accept `--lang ko`. When detected:
- Section headers and prose in Korean
- Technical terms remain in English
- Risk badges use Korean labels

## Risk Badges
| English | Korean |
|---------|--------|
| `[CRITICAL]` | `[심각]` |
| `[HIGH]` | `[높음]` |
| `[MEDIUM]` | `[중간]` |
| `[LOW]` | `[낮음]` |
